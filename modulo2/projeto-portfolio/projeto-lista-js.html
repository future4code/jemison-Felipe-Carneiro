PROJETO LISTA JS





Instruções de como o projeto é realizado:

O projeto lista JS é bem simples. A proposta é realizar os exercícios pedidos, realizando a montagem dos códigos em JavaScript e quando a resposta estiver correta, o mesmo será informado e irá passar para o próximo exercício. Obs: só conseguirá passar para a próxima etapa se estiver com a resposta correta. Acima temos o exemplo de quando a resposta está correta e sendo informado que passou naquele teste.

Tecnologias utilizadas para criação do projeto lista JS:
